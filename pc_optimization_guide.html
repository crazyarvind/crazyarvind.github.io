PC Optimization Guide

Introduction

Growing up, I used really slow computers, which included the HP Pavilion G4 and Dell Inspiron N5010. Their specifications are mentioned below.

Device Processor RAM Storage
HP Pavilion G4 AMD A4-3300M 2GB DDR3 500GB HDD
Dell Inspiron N5010 Intel i5 650 4GB DDR3 500GB HDD

As you can see, these machines were really slow, however I optimized them using various methods described below, which allowed me to run games released before 2010, many popular softwares, etc.

Step 1: Backup Your Data

First of all, I highly recommend you to copy all your important data to a USB stick or a cloud service. This is because we are going to fully reset the PC, so we can install a new operating system on it, and optimize it for performance boost.

Step 2: Install A New Operating System

Tip: If you are still using a HDD, I highly suggest switching to an SSD, if it supports your system. Look up your laptop model on google and search for storage upgrades. This will make your experience very smooth.

Now when you have backed up your data, it is time to install a new operating system. This will get rid of the old programs, cache, etc. that you previous system had.

You have two options from here, which are switching to a Windows operating system or switching to a Linux distribution. For most people, I recommend switching to a Windows OS since it will allow you to play old games and run various softwares without much efforts. Read this article if you really want to switch to Linux.

We are going to install the Windows 10 LTSC(Long Term Servicing Channel) version, which removes many useless programs, giving your system more space and resources.

Installing Windows

  1. Go to this website and download the "Windows 10 LTSC" version.
  2. Go to this website and download the windows version of ventoy. This program allows you to make a USB stick bootable, and put an operating system on it.
  3. Install and run ventoy, plug in your USB stick(which must have atleast 8GB of storage), select the USB stick name in Ventoy, and hit "install".
  4. Drag and drop your Windows 10 LTSC iso file into your USB stick.
  5. Search up the "boot key" for your laptop/desktop on google. Restart your PC(while keeping your USB stick plugged in), and as it is booting up, spam your boot key.
  6. You will see an interface which allows you to select a bootable storage device. Use your keyboard to select your USB stick.
  7. The Ventoy interface will boot up. Select the Windows option and it will lead you to the Windows installation interface.
  8. Fill your details in the setup, select your drive on which you want to install Windows(your main drive), and install windows.

Congratulations! You have successfully installed Windows 10 LTSC on your system!

Step 4: Debloat Your System

Windows runs many services in the background which slow down your PC. We are going to use the Chris Titus Debloater Tool to remove them, along with unecessary softwares.

To download it, go to The Ultimate Windows Utility and follow the instructions given to run the software. Check the boxes according to your convenience, however I do NOT recommend removing microsoft store and microsoft edge, as they can lead to unexpected behaviours.

Step 5: Manual Optimization

There are some manual optimizations you can do, which will push your system to its best performance.

  • Go to power settings and switch to "Best Performance" if it is not already selected.
  • Go to power button controls and make sure that "pressing the power button will make my pc" is set to "shutdown".
  • Whenever you install a new program like discord, steam or spotify, go to the task manager settings by right clicking on your task bar and selecting task manager. There, go to "startup apps" and disable those programs, otherwise they will always run on startup, slowing down your PC.
  • When playing video games, select the lowest preset and 720p resolution, which will allow the game to run smoothly. You can experiment by increasing the settings a bit, if they work on your system.
  • While running a heavy program or a game, go to task manager>services. Find your program there, right click on it, click "set priority" and then "high". This way, Windows will allocate more resources to your software. This will not give a huge boost, but it is good to do it.

Bonus: I highly recommend you to have atleast 8GB of RAM. Look up your laptop model on google and see if you can upgrade it, and which speed and generation is supported. You can find tutorials on youtube for your exact model on doing the RAM upgrade. For PCs, every hardware component is upgradable as long as it is appropriate with the motherboard.
